# 과제1 - 기본 FastAPI APP 구현

## 학습 목표

- FastAPI 프레임워크의 기본 개념과 구조를 이해한다.
- RESTful API를 설계하고 구현한다.
- Pydantic을 활용한 데이터 검증을 구현한다.
- 다양한 FastAPI 고급 기능을 활용한다.
